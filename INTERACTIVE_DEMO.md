**操作：**
1. 点击右上角的 **"Connect Wallet"** 按钮
2. MetaMask 会弹出连接请求
3. 点击 **"下一步"** → **"连接"**

**你应该看到：**
- 按钮变成 "Connecting..."
- MetaMask 弹窗

**解说词：**
```
"现在让我们连接钱包。点击 'Connect Wallet' 按钮。

系统会自动检测 MetaMask，并确保我们在正确的网络上。
如果不在 Mantle Sepolia，系统会自动提示切换。"
```

**连接成功后：**
- 显示钱包地址：`0xBebF...8f156`
- 绿色指示灯显示已连接
- 按钮变成 "Disconnect"

### 📍 步骤 4：匿名投票演示

**操作：**
1. 点击第一个提案的 **"Vote Anonymously"** 按钮

**弹窗内容：**
- 标题："Vote Anonymously"
- 提案标题："Proposal #1: Community Treasury Allocation"
- 提案描述
- 三个投票选项：
  - **Against** (红色)
  - **For** (绿色)
  - **Abstain** (灰色)
- 隐私说明
- Cancel / Submit Vote 按钮

**解说词：**
```
"点击投票按钮后，看到匿名投票界面。

这里有几个重要设计：
1️⃣ 三个选项：支持、反对、弃权
2️⃣ 选择后高亮显示
3️⃣ 明确告知：你的隐私受保护
4️⃣ 显示当前投票地址

这种设计确保用户理解：
✅ 投票是匿名的
✅ 选择是保密的
✅ 但结果可验证"
```

**选择并提交：**
1. 点击 **"For"** (绿色选项)
2. 点击 **"Submit Vote"**

**应该发生：**
- 按钮显示 "Submitting..."
- 2秒后弹出："✓ Vote submitted successfully! (Demo mode)"
- 弹窗关闭
- 投票统计更新

**解说词：**
```
"现在选择 'For'（支持）并提交。

在实际部署中，这里会：
1. 生成 Merkle Proof 证明投票资格
2. 生成 Nullifier 防止双投
3. 将加密的投票提交到链上
4. Gas 费约 $0.001

演示模式可以看到成功提示。
在实际使用中，这个过程在 MetaMask 中完成。"
```
